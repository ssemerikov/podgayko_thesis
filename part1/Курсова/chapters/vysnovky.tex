\section*{\centering ВИСНОВКИ}\addcontentsline{toc}{section}{ВИСНОВКИ}

У курсовій роботі проведено порівняльний бібліометричний аналіз досліджень моніторингу професійно-технічної освіти на основі 1998 публікацій бази Web of Science Core Collection за період 1997--2025 років. Аналіз виконано двома незалежними інструментами --- VOSViewer та аналітичною системою Python --- з подальшим кількісним порівнянням результатів.

\textbf{За результатами дослідження отримано такі основні висновки:}

1. Описова статистика (розділ~\ref{sec:descriptive_stats}) показала стабільне зростання кількості публікацій із моніторингу ПТО з піком у 2019--2023 роках. Географічно дослідження зосереджені в Європі (Німеччина, Нідерланди, Велика Британія), що відображає розвиненість систем дуальної освіти в цих країнах. Провідними науковими джерелами є спеціалізовані журнали з професійної освіти.

2. Бібліометричний аналіз мережі ключових слів у VOSViewer (розділ~\ref{sec:vos_network}) виявив 4 тематичні кластери: <<Системи та політика ПТО>> (11 термінів), <<Результати навчання>> (8), <<Педагогічна практика>> (6), <<Когнітивні результати>> (3). Центральними термінами є \textit{vet} (TLS\,=\,17\,857), \textit{study} (14\,924), \textit{student} (14\,844), \textit{training} (13\,158). Динаміка поля свідчить про зростання уваги до ролі роботодавців (\textit{company}, 2019,85) та контекстуалізованого навчання (\textit{context}, 2019,09).

3. Порівняльний аналіз (розділ~\ref{sec:chapter3}) продемонстрував високу відтворюваність кількісних метрик: кореляція частот входжень $r = 0{,}923$, Total Link Strength $r = 0{,}919$, середнього року публікації $r = 0{,}74$, середньої цитованості $r = 0{,}74$, нормалізованої цитованості $r = 0{,}79$.

4. Кластеризація алгоритмом Лувена в Python оптимально дає 3 кластери (ARI\,=\,0,47; NMI\,=\,0,61), об'єднуючи кластери <<Педагогічна практика>> та <<Когнітивні результати>> VOSViewer. Це пояснюється майже повною зв'язністю мережі (378/378 пар) та відмінностями в алгоритмах кластеризації.

\textbf{Обмеження дослідження:}
\begin{itemize}
    \item використання лише однієї бази даних (Web of Science);
    \item обмеження англомовними публікаціями;
    \item різний обсяг вибірок для VOSViewer (1000 записів) та Python (1998 записів), що впливає на абсолютні значення метрик;
    \item текстовий аналіз (text mining) охоплює лише заголовки та абстракти, а не повні тексти.
\end{itemize}

\textbf{Перспективи подальших досліджень.} Отримані результати створюють методичну основу для бакалаврської кваліфікаційної роботи автора <<Розробка інформаційно-аналітичної системи моніторингу й аналізу освітньої діяльності закладів професійної освіти>>. Перспективними напрямами є:
\begin{itemize}
    \item розширення аналізу на бази Scopus та ERIC для отримання повнішої картини;
    \item включення україномовних та інших неангломовних публікацій;
    \item розробка веб-інтерфейсу для інтерактивної візуалізації бібліометричних даних;
    \item інтеграція бібліометричного модуля в інформаційно-аналітичну систему моніторингу ПТО.
\end{itemize}
