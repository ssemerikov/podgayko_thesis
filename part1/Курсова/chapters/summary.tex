\section*{\centering \MakeUppercase{Анотація}}

\textit{Вступ.} Моніторинг освітньої діяльності закладів професійно-технічної освіти (ПТО) є актуальним завданням управління якістю освіти. Зростання кількості наукових публікацій із цієї тематики потребує систематичного бібліометричного аналізу для виявлення тематичної структури, ключових тенденцій та динаміки дослідницького поля. Метою роботи є порівняльний бібліометричний аналіз досліджень моніторингу ПТО засобами VOSViewer та Python.

\textit{Методи.} Проаналізовано 1998 публікацій бази Web of Science Core Collection (1997--2025). У VOSViewer проведено текстовий аналіз з бінарним підрахунком (поріг $\geq$100, тезаурус, 28 ключових слів). Для порівняння розроблено аналітичну систему з 9 Python-скриптів (NetworkX, scikit-learn, matplotlib), що відтворює аналіз частот, ко-входжень, кластеризації (алгоритм Лувена) та мережевих візуалізацій. Порівняння виконано за кореляцією Пірсона, ARI та NMI.

\textit{Результати.} VOSViewer виявив 4 тематичні кластери: <<Системи та політика ПТО>> (11 термінів), <<Результати навчання>> (8), <<Педагогічна практика>> (6), <<Когнітивні результати>> (3). Центральними термінами є vet (TLS\,=\,17\,857), study, student, training. Порівняльний аналіз показав високу кореляцію частот ($r = 0{,}923$) та TLS ($r = 0{,}919$). Кластеризація Python оптимально дає 3 кластери (ARI\,=\,0,47; NMI\,=\,0,61), об'єднуючи кластери 3 та 4.

\textit{Висновки.} Аналітична система Python успішно відтворює кількісні метрики VOSViewer ($r > 0{,}9$), проте має обмеження в точності кластеризації через різницю алгоритмів та повну зв'язність мережі. Результати створюють методичну основу для розробки інформаційно-аналітичної системи моніторингу ПТО.

\textbf{Ключові слова:} професійно-технічна освіта, бібліометричний аналіз, моніторинг, VOSViewer, Python, мережа ключових слів, кластерний аналіз, ко-входження.
