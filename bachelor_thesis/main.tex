% Бібліографічний стиль: DSTU 8302:2015 («Бібліографічне посилання. Загальні
% положення та правила складання») формально відсутній у пакеті TeX Live 2021;
% biblatex-dstu не входить до дистрибутива. Використано gost-numeric як
% найближчий технічний еквівалент: алфавітне сортування, нумерована
% бібліографія, формат «Прізвище І.Б. Заголовок: тип. Місто: Видавництво,
% рік. Cтор. doi». Структурні відмінності DSTU 8302 vs gost-2008 (Russian
% origin, який є предком gost-numeric.bbx) є мінімальними і ручно
% коректуються в окремих записах. Альтернатива — встановлення biblatex-dstu
% з GitHub поза TeX Live 2021 — залишена як post-defense backlog.

sorting=abyrvalg,
style=gost-numeric,
language=auto,
autolang=other,
backend=biber]{biblatex}

\DeclareSortingTemplate{abyrvalg}{
    \sort{\field{presort}}
    \sort[final]{\field{sortkey}}
    \sort{
        \field{sortname}
        \field{author}
        \field{editor}
        \field{sorttitle}
        \field{title}
        \field{journaltitle}
    }
    \sort{\field{sortyear}\field{year}}
    \sort{
        \field{sorttitle}
        \field{author}
        \field{editor}
        \field{title}
        \field{booktitle}
        \field{journaltitle}
    }
}
